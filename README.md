# Malindi Cleaners

A modern cleaning service website inspired by the [Cleanna Elementor Template Kit](https://themeforest.net/item/cleanna-modern-cleaning-service-company-elementor-pro-template-kit/63420104).

## Pages

- **Home** — Hero, services, pricing, testimonials, CTA
- **About** — Company story, values, stats
- **Services** — Full service catalog
- **Pricing** — Plans and FAQ
- **Team** — Team member profiles
- **Testimonials** — Client reviews and ratings
- **Blog** — Articles and tips
- **FAQ** — Frequently asked questions
- **Contact** — Contact form and info
- **404** — Error page

## Quick Start

```bash
npm start
# or: python3 -m http.server 3000
```

Open [http://localhost:3000](http://localhost:3000)

Or open `index.html` directly in your browser — no server required.

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Outfit + DM Sans)
- Unsplash images (replace with your own)

## Customization

- Edit colors in `assets/css/style.css` (`:root` variables)
- Update contact info in each page's top bar and footer
- Replace images with your own branded photography
